# Minecraft Helper スキル

MinecraftコマンドサイトのためのGemini CLIカスタムスキルです。
コマンドの解説、生成、検証を効率的に行います。

## 機能

### 1. コマンド解説生成
指定されたMinecraftコマンドの詳細な解説を生成します。

**使用例:**
```
gemini "minecraft-helper: /giveコマンドの完全解説を生成して"
```

**出力内容:**
- コマンドの基本構文
- 各パラメータの説明
- 使用例（初級〜上級）
- よくある間違い
- 関連コマンド

### 2. コマンド生成
要望に基づいてMinecraftコマンドを生成します。

**使用例:**
```
gemini "minecraft-helper: ダイヤの剣にシャープネス5とアンブレイキング3を付けてプレイヤーに渡すコマンド"
```

**出力内容:**
- 完成したコマンド
- 各部分の解説
- コピペ用テキスト
- バージョン互換性情報

### 3. コマンド検証
ユーザーが入力したコマンドの正確性を検証します。

**使用例:**
```
gemini "minecraft-helper: このコマンドをチェック: /give @p diamond_sword{Enchantments:[{id:sharpness,lvl:5}]}"
```

**出力内容:**
- 構文エラーの有無
- 修正提案
- 最適化の提案
- バージョン互換性警告

### 4. NBTタグ生成
アイテムやエンティティのNBTタグを生成します。

**使用例:**
```
gemini "minecraft-helper: カスタム名と説明文を持つダイヤの剣のNBTタグ"
```

### 5. データパック支援
データパックやコマンドブロックシステムの設計を支援します。

**使用例:**
```
gemini "minecraft-helper: プレイヤーが特定のエリアに入ったら通知するシステム"
```

## 対応バージョン

| エディション | バージョン | 対応状況 |
|-------------|-----------|---------|
| Java Edition | 1.21.x | 完全対応 |
| Java Edition | 1.20.x | 対応 |
| Bedrock Edition | 最新 | 一部対応 |

## チェックポイント

### コマンド解説生成時
- [ ] 構文が正確か
- [ ] パラメータ説明が完全か
- [ ] 使用例が動作確認済みか
- [ ] バージョン情報が明記されているか

### コマンド生成時
- [ ] 要望を満たしているか
- [ ] 構文エラーがないか
- [ ] 最適化されているか
- [ ] コピペですぐ使えるか

### SEO対策
- [ ] 適切な見出し構造か
- [ ] 重要キーワードが含まれているか
- [ ] 分かりやすい日本語か
- [ ] 構造化データ用の情報があるか

## 注意事項

- Java EditionとBedrock Editionではコマンド構文が異なる場合があります
- バージョンによって使用可能なコマンドやパラメータが異なります
- NBTタグはJava Edition専用です
