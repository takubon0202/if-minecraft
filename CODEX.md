# Minecraftコマンドサイト - プロジェクト情報

このファイルはCodex CLI用のプロジェクトコンテキストを提供します。

## プロジェクト概要

Minecraftのコマンドを学習・生成できるWebサイトを開発しています。

### ターゲットユーザー
- **初心者**: コマンドを初めて使う人
- **中級者**: 基本は知っているが応用を学びたい人
- **上級者**: 効率的なコマンド生成ツールを求める人

### 主な機能

1. **コマンド解説**
   - /give, /summon, /execute, /tp 等の詳細解説
   - NBTタグの説明
   - ターゲットセレクターの使い方

2. **コマンドジェネレーター**
   - GUIでパラメータを選択
   - リアルタイムでコマンドを生成
   - コピーボタンでクリップボードに保存

3. **ツール**
   - NBTタグエディター
   - JSON Text Generator
   - 座標計算ツール

## 技術スタック

- **フロントエンド**: JavaScript/TypeScript
- **スタイル**: CSS/SCSS
- **フレームワーク**: （未定 - React/Next.js/Astro等を検討）

## ディレクトリ構成（予定）

```
src/
├── pages/           # ページコンポーネント
├── components/      # 共通コンポーネント
├── generators/      # コマンドジェネレーター
├── data/            # Minecraftデータ（アイテム、エンティティ等）
├── utils/           # ユーティリティ関数
└── styles/          # スタイルシート
```

## コーディング規約

- コメントは日本語で記述
- 変数名・関数名は英語（キャメルケース）
- セマンティックHTML使用
- アクセシビリティを考慮（ARIA属性等）
- SEO最適化（メタタグ、構造化データ）

## Minecraftバージョン

- **Java Edition**: 1.21.x（最新）
- **Bedrock Edition**: 対応予定

## CLI設定

### 使用モデル
- Codex: gpt-5.3-codex

### コマンド例

```bash
# 非対話モード（Claude Code連携用）
codex exec "summonコマンドのNBTタグ解説を生成"
codex exec "アイテムデータのJSONを作成"

# ヘルパースクリプト経由
node scripts/codex-helper.js --minecraft "giveコマンドジェネレーターを実装"
```

## 注意事項

- 使用量上限に達した場合は即座に報告し、使用を停止
- 追加課金を防ぐため、制限が回復するまで待機
