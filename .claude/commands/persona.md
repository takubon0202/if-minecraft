# ペルソナAI評価スキル - ユーザー視点での評価

想定ターゲットユーザー（ペルソナ）の視点でページを評価・採点します。

## 使用方法

```
/persona [ファイルパスまたはURL]
```

引数がない場合は、現在のプロジェクトの主要ページを評価します。

## ペルソナ設定（Minecraftコマンドサイト用）

### ペルソナ1: 初心者マインクラフター（太郎くん）
- **年齢**: 12歳
- **プレイ歴**: 3ヶ月
- **特徴**: コマンドを使ってみたいけど、難しそうで怖い
- **課題**: /giveや/summonの使い方が分からない
- **目標**: 友達に自慢できるようなクールなコマンドを覚えたい

### ペルソナ2: 中級プレイヤー（花子さん）
- **年齢**: 16歳
- **プレイ歴**: 2年
- **特徴**: 基本的なコマンドは知っているが、複雑なものは調べながら
- **課題**: NBTタグやターゲットセレクターがよく分からない
- **目標**: 配布マップやミニゲームを作りたい

### ペルソナ3: 上級者・配布者（けんじさん）
- **年齢**: 24歳
- **プレイ歴**: 8年
- **特徴**: コマンドブロックでシステムを組める
- **課題**: 最新バージョンの変更点や、より効率的な方法を知りたい
- **目標**: 質の高い配布マップを作成・公開したい

## 評価基準

### 1. 共感度 (35点満点)

- 自分の悩みが解決できそうと感じるか
- サイトの雰囲気が親しみやすいか
- 「このサイトは自分のためにある」と感じるか
- 使用している言葉が分かりやすいか
- ビジュアルが魅力的か

### 2. 通いたい度 (35点満点)

- ブックマーク・お気に入りに登録したいか
- 定期的にアクセスしたいコンテンツがあるか
- 他のページも見たくなるか
- 信頼できる情報源と感じるか
- コミュニティに参加したいと思うか

### 3. サービス利用意欲 (30点満点)

- 提供されているツールを使いたいか
- コマンドジェネレーターを試したいか
- 解説を読んで学びたいか
- 人に教えたい・共有したいか
- もっと高度な機能があれば使いたいか

## 合格基準

- **合格ライン**: 全ペルソナ平均70点以上
- **優秀ライン**: 全ペルソナ平均85点以上
- **不合格**: いずれかのペルソナで60点未満

## 実行例

```bash
# 特定ファイルを評価
/persona index.html

# URLを評価
/persona http://localhost:3000

# 特定のペルソナのみで評価
/persona --persona=初心者 index.html

# 全ページを評価
/persona --all
```

## 他のスキルとの連携
