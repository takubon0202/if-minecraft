# 総合レビュースキル - 評価＆ペルソナ統合レビュー

/evaluate と /persona を統合実行し、総合的なページ品質レビューを行います。
合格点に達するまで自動で改善を繰り返すことも可能です。

## 使用方法

```
/review [ファイルパスまたはURL]
```

## 実行フロー

1. **評価AI** による技術的評価（SEO、明確さ、分かりやすさ、申込意欲）
2. **ペルソナAI** によるユーザー視点評価（共感、通いたい度、利用意欲）
3. **統合スコア** の算出
4. **改善提案** の生成
5. （オプション）**自動修正** の実行

## 評価ウェイト

| 評価種別 | ウェイト |
|---------|---------|
| 評価AI（技術的評価） | 50% |
| ペルソナAI（ユーザー視点） | 50% |

## 合格基準

- **統合スコア 90点以上**: 合格（公開可能レベル）
- **統合スコア 95点以上**: 優秀（高品質）
- **統合スコア 90点未満**: 不合格（要修正）

## オプション

```bash
# 基本実行
/review index.html

# 自動修正モード（合格点に達するまで改善を繰り返す）
/review --auto-fix index.html

# 最大繰り返し回数を指定
/review --auto-fix --max-iterations=5 index.html

# レポートをファイルに出力
/review --output=report.md index.html

# 簡易モード（スコアのみ表示）
/review --quick index.html
```

## 使用例

```bash
# 開発中のページを総合レビュー
/review src/pages/index.html

# 本番公開前の最終チェック
/review --auto-fix dist/index.html

# 全ページの一括レビュー
/review --all src/pages/
```
